# Listing Description Generator

A Python script that uses the Claude API to generate 3 real estate listing descriptions in different tones (luxury, friendly, investment) from a single property input.

## Tools
- Python
- Anthropic Claude API (claude-opus-4-6)

## How it works
Input a property address, bedrooms, bathrooms, and key features → Claude outputs 3 listing descriptions in luxury, friendly, and investment tones.
