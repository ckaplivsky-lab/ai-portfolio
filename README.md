# Listing Description Generator

A Python script that uses the Claude API to generate 3 real estate listing descriptions in different tones from a single property input.

## Tools
- Python
- Anthropic Claude API

## How it works
Input a property address, bedrooms, bathrooms, and key features. Claude outputs 3 descriptions:
- Luxury: elegant and aspirational
- Friendly: warm and family-oriented  
- Investment: facts-focused and ROI-driven
